真正的自由无限制

使用任何作业系统在校园连接互联网

无论你使用 Windows(R), GNU/Linux，还是其他 UNIX(R) 作业系统（如 FreeBSD，DragonFly BSD 等等等），你都可以轻易接入互联网，互联网服务供应商提供的软件（Netkeeper）仅支持级少量的作业系统（通常只有 Windows 和 macOS），这迫使学生不得不花费巨资购买价格高昂的 Windows 软件许可证或苹果电脑。Seth 软件可以帮助学生打破大资本家的垄断。

Seth 与 FOSS 软件兼容

Seth 软件包和 SDK 是以麻省理工学院许可证（MIT License）许可的自由&开源软件。互联网服务供应商提供的软件（Netkeeper）与 FOSS 软件不兼容。

在 Intel/AMD 之外的处理器上使用

Seth 软件包和 SDK 可以在非 x86 家族的处理器上运行。互联网服务供应商提供的软件（Netkeeper）只能在 x86 家族的处理器上运行，这使学生在购买计算机时只能考虑昂贵的 Intel/AMD 处理器。现在，市场上存在大量廉价的基于 ARM 或者 MIPS （以后可能还有 RISC-V）SoC 的计算机。使用 Seth，你就无需在购买计算机时受到不公正的待遇。

速度翻倍

想要等待连接网络的时间更短？这里有。Seth 现在达到比供应商软件的数十倍快。

Seth 的桌面版软件经过专门的优化设计，速度比供应商软件启动快数十倍。使用 Seth，联网快人一步，节省更多的时间进行你的学业、科研、实验工作。赢在起跑线。

比供应商软件更轻

更少内存占用意味着能给您电脑流畅运行腾出更多空间。电脑上其他程序将倍感轻松。

Seth 的桌面版软件无需在系统后台持续运行。免除了因不必要的上下文切换和后台任务带来的 CPU 开销。此外，更少的内存使用 = 可以启动更多的应用程序。让你的电脑发挥它的最大价值。

等等，如果你是开发者，Seth 还是一个通用的平台

Seth 提供了一个开放的、通用的框架以开发应用程序

Seth 库和框架使得制作自己需要访问 Netkeeper 功能的应用程序如此简单。

Seth-Desktop（Seth 的桌面版软件）高度可定制

Seth 的桌面版软件提供多种后端（如 RP-PPPoE，RP-PPPoE 内核模式，NetworkManager 集成、Win32 RAS API、以及将来要添加的 Webhook 等）和多种前端（CLI 交互模式、CLI 脚本模式、GTK+ 图形界面模式）。你可以轻易创建前端或后端而不影响其他模块，同时可以重用了大量代码，快速与你的业务集成。

数据文件即将永久免费提供，我们正在处理一些基础设施方面的问题

“Seth NK 数据文件”：为了防止被恶意利用，Seth 数据文件的生成工具不向最终用户分发。必须利用数据文件，用户才可以使用 Seth 软件或基于 Seth 的其他工具从数据文件中获取必要的信息，并完成 Netkeeper 4.11.x 拨号。”数据文件“具有几个特性： 一个”数据文件“只能在一段时间内工作。一个”数据文件“只对一个 Netkeeper 账户（可能是你的手机号）有效，不同人的”数据文件“不能通用。（类似于 Linux 上很多驱动程序，驱动程序开源，但固件不开源）。数据文件只是“数据”而不是可执行代码，也无法影响 Seth 软件的逻辑，Seth 软件 100% 与 FOSS 兼容。
